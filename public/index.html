Welcome to Task Manager Api

A modern task manager api tool to handle different types of operations:-

https://samar-task-manager-api.herokuapp.com

    This route can be used to create a new user
  • sendhttps://samar-task-manager-api.herokuapp.com/users/login
    This route can be used to login a user
  • sendhttps://samar-task-manager-api.herokuapp.com/users/logout
    This route can be used to logout a user
  • sendhttps://samar-task-manager-api.herokuapp.com/users/me
    This route can be used to know about a user
  • sendhttps://samar-task-manager-api.herokuapp.com/tasks
    This route can be used to create a new task
  • sendhttps://samar-task-manager-api.herokuapp.com/tasks
